# **最後的戰役FinalBattle**
2087年在地球發生了一場全球性的襲擊  
戰場遍佈在地球的各個角落  
臺灣地區指揮部派遣ASSC前往受到襲擊的現場進行偵查行動  
## **遊戲玩法**
### **第一關**
**滑鼠**  
移動滑鼠 瞄準  
左鍵 射擊  
**鍵盤**  
WASD或方向鍵 操縱角色移動  
B 切換射擊模式  
### **第二關**
**鍵盤**  
WASD或方向鍵 操縱角色移動  
空白鍵 射擊  
B 切換武器  
### **第三關**
**滑鼠**  
移動滑鼠 瞄準  
左鍵 射擊  
**鍵盤**  
WASD或方向鍵 操縱角色移動  
B 切換射擊模式
